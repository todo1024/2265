课程介绍：

CCNA认证标志着具备安装、配置、运行中型路由和交换网络，并进行故障排除的能力。获得CCNA认证的专业人士拥有相应的知识和技能，能够通过广域网与远程站点建立连接，消除基本的安全威胁，了解无线网络接入的要求。CCNA培训包括（但不限于）以下这些协议的使用：IP、EIGRP、串行线路接口协议、RIPv2、VLAN、以太网和访问控制列表（ACL）
